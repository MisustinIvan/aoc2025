# Solutions for Advent of Code 2025 in elixir

## Getting started
To get started, create the `./priv/inputs/` folder and for each day create a `day<nn>.txt`.

Then, to run a specific solution, run `mix aoc <day>`.

You can also provide an optional argument `--part 1|2|both`.

## Testing
To run the unit tests, run `mix test`.
